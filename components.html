Components

The following components are the tools included in the Fort library.

Popup fort-popup.js

To use the Popup, include the required javascript and css files, and initiate it by calling one of the following:

Download

The jQuery plugin

	$(".showPopup").popup();

Creating a javascript object

	var p = new Popup(".showPopup");

Note: The class .showPopup can be replaced with any class name, as long as the class used in the markup matches the class used in initiating the popup.

Example

Show Another Popup!

Html

    <button id="menu1" class="showPopup">Show Popup!</button>
    <button id="menu2" class="showPopup">Show Another Popup!</button>

Javascript

    var popup = $(".showPopup").popup();
    popup.addMenu("menu1", "Hello World!",
        [
            {"name": "Menu Item 1", url: "m1"},
            {"name": "Menu Item 2", id: "m2"},
            {"name": "Link to second button's menu...", id: "menu2"}
        ]
    );
    popup.addMenu("menu2", "Hello Again, World!",
            [
                {"name": "Another Menu Item 1", url: "am1"},
                {"name": "Another Menu Item 2", url: "am2"},
                {"name": "Link to first button's menu...", id: "menu1"}
            ]
    );
