Components

The following components are the tools included in the Fort library.

Popup fort-popup.js

Download

To use the Popup, first include the required javascript and css files.

Next, create the popup with either a jQuery plugin call, or by creating a classic javascript object:

The jQuery plugin

    var menu1 = {
        id: "menu1", title: "Hello World",
        contents: [
            {name: "Item 1"},
            {name: "Item 2"},
            {name: "Item 3"}]
        });
    $("#menu1").popup(menu1);

Creating a javascript object

    var p = new Popup("#menu1");
    p.addMenu(menu1);

Note: The id #menu1 is also the name linked to the first menu for that button. To access the starting menus from other buttons, just set the menu item's id to the starting menu's id.

Example

Show Another Popup!

Html

    <button id="menu1">Show Popup!</button>
    <button id="menu2">Show Another Popup!</button>

Javascript

    $("#menu1").popup({
        id: "menu1",
        title: "Hello World!",
        contents: [
            {"name": "Menu Item 1", url: "m1"},
            {"name": "Menu Item 2", id: "m2"},
            {"name": "Link to second button's menu...", id: "menu2"}
        ]
    });
    $("#menu2").popup({
        id:"menu2",
        title: "Hello Again, World!",
        contents: [
                {"name": "Another Menu Item 1", url: "am1"},
                {"name": "Another Menu Item 2", url: "am2"},
                {"name": "Link to first button's menu...", id: "menu1"}
            ]
        }
    );

Events

The following are jQuery custom events fired at different times within the popup:

Event Description Objects Passed
popup.created This event is fired at the end of the createPopup function. None
popup.resize This event is fired when the window is resized (redundant event, will be removed in future release). None
popup.setContent This event is fired at the end of the setContent function, showing the contents have been modified. None
popup.visible This event is fired at the end of the toggleVisible function and fires when the popup is visible to the user. (Currently does not wait css transitions to complete). None
popup.menuClicked This event is fired when a menu item in the popup has been clicked. None

Popup w/ jScrollPane Plugin

The popup can be easily integrated with the jScrollPane jQuery plugin. This adds mobile functionality to scrolling divs and adds more customizability to the look and feel of the popup.

Javascript

	//To initialise, use the code below:
	$(document).on("popup.created", function () {
		$("#popupContentWrapper").jScrollPane({
			horizontalGutter: 0,
			verticalGutter: 0,
			'showArrows': false
		});
	});

	$(document).on('popup.setContent popup.visible popup.resize', function (e) {
		$("#popupContentWrapper").data('jsp').reinitialise();
	});
