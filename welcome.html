About

Fort is a library full of Fundamental Open-source Responsive Tools. The components were designed with mobile browsing in mind and should work on Android 2.3+, iOS4+, and many others.

Popup Examples

Html

    <button id="button1" class="showPopup">Show Popup!</button>
    <button id="button2" class="showPopup">Show Another Popup!</button>

Css

    var popup = $(".showPopup").popup();
    popup.addMenu("button1", "Hello World!",
        [
            {"name": "Menu Item 1", url: "m1"},
            {"name": "Menu Item 2", id: "m2"},
            {"name": "Link to second button's menu...", id: "button2"}
        ]
    );
    popup.addMenu("button2", "Hello Again, World!",
            [
                {"name": "Another Menu Item 1", url: "am1"},
                {"name": "Another Menu Item 2", url: "am2"},
                {"name": "Link to first button's menu...", id: "button1"}
            ]
    );
